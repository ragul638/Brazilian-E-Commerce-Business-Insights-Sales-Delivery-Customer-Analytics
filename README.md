# 📊 Brazilian E-Commerce Data Analysis Project

## 🔍 Problem Statement

This project analyzes a real-world Brazilian e-commerce dataset to uncover insights related to customer behavior, delivery performance, revenue trends, and product performance. The goal is to identify key business challenges and provide data-driven recommendations.

---

## 🛠 Tools & Technologies Used

* SQL (Data Cleaning & Joins)
* Python (Pandas, NumPy, Matplotlib, Seaborn for EDA)
* Power BI (Interactive Dashboard & Visualization)

---

## 📁 Dataset

* Source: Olist Brazilian E-Commerce Dataset (Kaggle)
* Contains multiple tables including customers, orders, payments, products, sellers, and reviews.

---

## ❓ Key Business Questions

* Which states generate the highest revenue?
* What is the percentage of repeat vs one-time customers?
* How do delivery delays impact customer satisfaction?
* Which product categories drive the most revenue?
* What is the cancellation rate and revenue loss?

---

## ⚙️ Project Workflow

### 1. Data Cleaning (SQL)

* Cleaned data
* Handled missing values and duplicates
* Joined multiple tables using keys
* Created a final structured dataset

### 2. Data Analysis (Python)

* Cleaned data deeply 
* Performed Exploratory Data Analysis (EDA)
* Created new features like delivery time and delay
* Identified trends, correlations, and outliers

### 3. Data Visualization (Power BI)

* Built interactive dashboards with:

  * Sales Overview
  * Delivery Performance
  * Customer & Product Insights

---

## 📊 Key Insights

* Revenue is highly concentrated in major urban states
* Late deliveries negatively impact customer review scores
* Certain product categories have high freight costs relative to revenue
* A large portion of customers are one-time buyers
* Specific sellers consistently underperform in delivery timelines

---

## 📈 Dashboard Highlights

* Sales trends and revenue breakdown
* Delivery delay analysis by region
* Customer behavior and seller performance

---

## 🚀 Conclusion

This project demonstrates how data can be used to identify operational inefficiencies and improve customer satisfaction in an e-commerce business.
